Boostrap 4.5 - Introduction

Cette page fonctionne avec Boostrap 4 en CDN

Présentation générale

  • Boodstrap est un framework CSS "cadriciel"
  • C'est une librairie coté client (front-end)
  • Boostrap nous aide à mettre en forme les pages web.
  • C'est une librairie CSS 3 mais il y a également des composants HTML 5 et JS.
  • Il est basé sur SASS (css avancé) et normalize
  • Il comporte également une grille de mise en pages

Interêts ?

  • Uniformisation pour les navigateurs
  • Gain de temps
  • Styles identiques
  • La grille responsive !!!
  • Plein de suppléments en +
  • C'est gratuit

Inconvénients ?

Cela suppose un temps d'apprentissage

Utilisation importante des div, donc bien se repérer dans son code
